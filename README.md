# Maylaa International - Asset Management System

A professional, luxury-themed asset management application designed to replace Excel-based tracking with a secure, automated SQL-powered web system.

## ✨ Features

- **Enterprise Analytics Dashboard**: Visual charts for category distribution and entity-wise asset allocation.
- **Entity Switching**: Seamless navigation across multiple business branches (Touch of Oud, Karji, etc.).
- **Global Multi-Sheet Export**: Generate professional Excel reports where each Entity has its own tab.
- **Excel & Clipboard Import**: Bulk upload data directly from Excel files or copy/paste directly from the clipboard.
- **Real-time CRUD Ops**: Add, Edit, Soft-Delete, and Assign assets with immediate database persistence.
- **Luxury Aesthetic**: A bespoke glassmorphism UI themed around premium perfume and chocolate branding.

## 🚀 Tech Stack

- **Frontend**: React (Vite), Lucide Icons, Recharts.
- **Backend**: Node.js, Express.
- **Database**: SQLite (SQL Engine).
- **Styling**: Vanilla CSS (Custom tokens and micro-animations).

## 🛠️ Setup Instructions

### 1. Backend Server
```bash
cd AB
npm install
node server.js
```
The server will initialize a local SQL database in `asset-data/database.sqlite`.

### 2. Frontend Application
```bash
cd AB
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.
